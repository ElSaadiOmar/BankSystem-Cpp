# Bank Management System

A fully functional console-based bank management system built in **C++** as a capstone project for Part One of the ProgrammingAdvices roadmap.

## Features
- Customer account management (add, update, delete, search)
- Deposit, withdrawal, and transfer operations
- convert currency system
- User management with login and permissions system
- Persistent file-based storage (no database required)

## Technologies
- Language: C++
- Paradigm: Object-Oriented Programming (OOP)
- Storage: File-based persistence

## What I Learned
- Applying OOP principles: encapsulation, class hierarchies, modular design
- Managing file I/O for persistent data storage
- Building a multi-screen console application with user/permission management
